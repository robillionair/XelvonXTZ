What is the real ANANSI product?+

ANANSI is a sovereign memory and context system in development. It is intended to preserve approved knowledge, govern access and prepare portable context for a selected AI model.

Is ANANSI available now?+

Not yet. ANANSI is in active development. The early-access list provides updates and possible invitations to controlled testing; joining does not guarantee selection.

Does ANANSI remember everything?+

No. That idea belongs to the fictional universe. The real product is being designed to use only approved sources with explicit permissions, inspection and removal controls.
